### 3. 数据验证

当有数据后，可以通过以下方式验证：

```sql
-- 查看每天的作业记录
SELECT work_date, COUNT(*) as count, SUM(acre) as total_acre
FROM work_records
WHERE source = 'old_api'
GROUP BY work_date
ORDER BY work_date DESC
LIMIT 30;

-- 查看各设备的作业情况
SELECT macid, cooperative_name, work_type_name, 
       COUNT(*) as record_count, SUM(acre) as total_acre
FROM work_records wr
JOIN old_supplier_devices osd ON wr.t_number = osd.macid
WHERE wr.source = 'old_api'
GROUP BY macid
ORDER BY total_acre DESC;
```
