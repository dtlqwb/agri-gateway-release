| 中文列名 | 英文字段 | 说明 | 示例 |

| 机具类型 | machine_type | 农机具类型 | 拖拉机、收割机等 |
| 备注 | remark | 其他说明信息 | - |

## 📊 数据格式要求

### 1. 日期格式
支持以下格式：
- `2024-05-15` (推荐)
- `2024/05/15`
- `2024年5月15日`
- Excel日期格式（自动转换）

### 2. 数值格式
- 面积：数字，可以是小数（如 125.5）
- 时长：数字，单位为**分钟**
- 不要包含单位符号（如 "125.5亩" ❌，应该是 125.5 ✅）

### 3. 文本格式
- 设备号：纯数字或字母数字组合
- 姓名：中文字符
- 合作社：任意文本

## ⚠️ 注意事项

### 1. 必填项检查
- **设备号**和**作业日期**是必填的，缺少会导致该行被跳过
- 如果只有车牌号没有设备号，系统会自动生成临时设备号

### 2. 数据去重
- **当前版本不会自动去重**，请在导入前手工检查
- 避免重复导入相同的数据

### 3. 文件大小限制
- 最大文件大小：**10MB**
- 建议单次导入不超过5000行数据

### 4. 文件格式
- 支持 `.xlsx` 和 `.xls` 格式
- 不支持 `.csv` 格式（需要转换为Excel格式）

### 5. 特殊字符
- 避免在文本字段中使用特殊字符（如 `<>"/\|?*`）
- 合作社名称中的 `/` 会被保留

## 🚀 导入步骤

### 方法1：通过Web界面导入（推荐）

1. **登录系统**
   - 访问 http://localhost:3001
   - 使用管理员账号登录

2. **进入导入页面**
   - 点击左侧菜单"数据管理" → "Excel导入"

3. **上传文件**
   - 点击"选择文件"按钮
   - 选择准备好的Excel文件
   - 点击"预览"查看数据

4. **确认列映射**
   - 系统会自动识别列名
   - 检查映射是否正确
   - 如有错误，手动调整列映射

5. **确认导入**
   - 点击"确认导入"按钮
   - 等待导入完成
   - 查看导入结果统计

### 方法2：通过API导入

```bash
curl -X POST http://localhost:3001/api/import/confirm \
  -H "Authorization: YOUR_TOKEN" \
  -F "file=@your_file.xlsx" \
  -F "importType=work_records"
```

## ❓ 常见问题

### Q1: 导入时提示"列映射失败"怎么办？
**A**: 检查Excel表头是否包含系统能识别的列名。参考上面的"支持的列名变体"列表，或者手动在导入页面调整列映射。

### Q2: 部分数据行被跳过了？
**A**: 可能的原因：
- 缺少设备号或作业日期
- 数据格式不正确（如日期格式错误）
- 查看导入结果中的错误信息，定位具体问题行

### Q3: 如何批量修改设备号？
**A**: 如果旧数据没有设备号，可以：
1. 先用Excel的公式生成设备号
2. 或使用车牌号作为设备号（系统会自动生成）

### Q4: 导入后发现数据有误，如何修正？
**A**: 
1. 在"数据修复"页面手动修改
2. 或删除错误数据后重新导入

### Q5: 一次可以导入多少数据？
**A**: 
- 文件大小不超过10MB
- 建议单次不超过5000行
- 大量数据建议分批导入
