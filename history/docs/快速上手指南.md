# 智慧农机数据聚合平台 - 快速上手指南

## 📦 安装包内容

您收到的 `agri-gateway-v3.0.zip` 包含完整的项目文件。

---

## 🚀 5分钟快速启动

### 第1步：解压文件

将压缩包解压到任意目录，例如：
```
D:\agri-gateway-release\
```

### 第2步：安装依赖

打开命令行（CMD或PowerShell），进入项目目录：

```bash
cd D:\agri-gateway-release
npm install
```

等待依赖安装完成（首次可能需要几分钟）。

### 第3步：配置数据库

确保您的电脑上已安装 **MySQL 8.0**。

创建配置文件 `.env`：

```bash
# 复制模板
copy .env.example .env
```

编辑 `.env` 文件，修改以下配置：

```bash
# MySQL数据库配置（根据实际情况修改）
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=你的MySQL密码
MYSQL_DATABASE=agri_gateway

# Token密钥（生成一个随机字符串）
# 运行命令: node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
TOKEN_SECRET=粘贴生成的随机字符串

# 云途安API配置（如果需要同步云途安数据）
YT_USERNAME=your_username
YT_PASSWORD=your_password

# 服务端口（默认3001，如冲突可修改）
PORT=3001
```

**生成TOKEN_SECRET的命令**：
```bash
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

### 第4步：启动服务

```bash
npm start
```

看到以下输出表示启动成功：

```
🚀 聚合平台已启动: http://localhost:3001
   API: http://localhost:3001/api/health
   前端: http://localhost:3001
```

### 第5步：访问系统

打开浏览器访问：**http://localhost:3001**

**默认管理员账号**：
- 用户名：`admin`
- 密码：`admin123`

**登录后请立即修改密码！**

## ❓ 常见问题

### Q1: 启动时报错"Cannot find module"？
**A**: 依赖未安装，运行 `npm install`

### Q2: 提示"数据库连接失败"？
**A**: 检查`.env`中的MySQL配置是否正确，确保MySQL服务已启动

### Q3: 端口3001被占用？
**A**: 修改`.env`中的`PORT`为其他端口，如3002

### Q4: 首次启动很慢？
**A**: 首次启动会初始化数据库并执行全量同步，需要几分钟，属正常现象

### Q5: 如何备份数据？
**A**: 定期备份MySQL数据库：
```bash
mysqldump -u root -p agri_gateway > backup.sql
```
