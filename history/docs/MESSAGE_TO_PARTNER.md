关于端口号的问题，我解释一下：

## 📌 情况说明

项目默认端口是 **3000**，但我推荐配置为 **3001**。

你看到 3000 端口，可能是因为：
- 没有创建 `.env` 配置文件
- 或者 `.env` 文件中没有配置 `PORT` 变量

## ✅ 解决方法（二选一）

### 方法1: 改为 3001 端口（推荐）

1. 在项目根目录，复制配置文件：
   ```bash
   # Windows
   Copy-Item .env.example .env
   
   # Linux/Mac
   cp .env.example .env
   ```

2. 打开 `.env` 文件，确认这一行存在：
   ```
   PORT=3001
   ```

3. 重启服务：
   ```bash
   node index.js
   ```

现在应该显示：`http://localhost:3001`

### 方法2: 继续使用 3000 端口（也可以）

如果你觉得 3000 没问题，可以继续使用，只需访问时输入：
```
http://localhost:3000
```

**注意**: 使用 3000 端口的话，所有文档中的 3001 都要改成 3000。

## 💡 建议

推荐使用 **3001 端口**，因为：
- 避免与 Node.js 默认端口冲突
- 与项目文档保持一致
- 便于区分不同环境
