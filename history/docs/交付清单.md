## 🎯 v3.0 主要更新

### 安全加固
- 🔐 HMAC-SHA256 Token认证（防伪造、防篡改）
- 🔐 24小时Token自动过期
- 🔐 磁盘存储替代内存存储（避免OOM）
- 🔐 文件上传限制10MB，仅允许Excel格式
- 🔐 自动清理临时文件
- 🔐 完善的敏感信息保护（.env不被提交）

### 功能优化
- ✨ Excel导入性能提升（移除去重逻辑）
- ✨ 支持land_group字段
- ✨ 更灵活的列名识别
- ✨ 详细的导入错误提示

## 📤 发送给同事时需要提供

### 1. 安装包
- `agri-gateway-v3.0.zip`

### 2. 必需的环境（同事需要自行安装）
- Node.js >= 14.x
- MySQL >= 8.0
- （可选）Playwright浏览器（用于旧供应商数据抓取）

### 3. 配置信息（需要单独告知）
```bash
# MySQL数据库信息
MYSQL_HOST=localhost
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=【需要提供】
MYSQL_DATABASE=agri_gateway

# 云途安API账号（如需要同步）
YT_USERNAME=【需要提供】
YT_PASSWORD=【需要提供】

# 旧供应商账号（如需抓取）
OLD_SUPPLIER_USERNAME=【需要提供】
OLD_SUPPLIER_PASSWORD=【需要提供】
```

### 4. 重要提醒
⚠️ **首次启动后必须**：
1. 生成并配置TOKEN_SECRET
2. 修改默认管理员密码（admin/admin123）
3. 不要将.env文件分享给他人

## 🚀 同事使用步骤（简要）

1. 解压 `agri-gateway-v3.0.zip`
2. 运行 `npm install` 安装依赖
3. 复制 `.env.example` 为 `.env` 并配置
4. 运行 `npm start` 启动服务
5. 访问 http://localhost:3001
6. 使用 admin/admin123 登录
7. **立即修改密码**

详细说明请查看：**快速上手指南.md**
