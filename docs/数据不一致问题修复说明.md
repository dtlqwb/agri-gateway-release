# 数据不一致问题修复说明

**修复时间**: 2026-04-18  
**修复内容**: 修复 `repairMachineData()` 函数，支持 `old` 和 `old_api` 两种数据源

---

## 🔧 修复内容

### 修改文件: `services/db.js`

**函数**: `repairMachineData()` (第452-505行)

**修改点**:

1. **查询缺失设备时** (第461行)
   ```javascript
   // 修改前
   WHERE wr.source = 'old'
   
   // 修改后
   WHERE wr.source IN ('old', 'old_api')
   ```

2. **检查是否已存在时** (第463行)
   ```javascript
   // 修改前
   AND wr.t_number NOT IN (SELECT t_number FROM machines WHERE source = 'old')
   
   // 修改后
   AND wr.t_number NOT IN (SELECT t_number FROM machines WHERE source IN ('old', 'old_api'))
   ```

3. **更新年度面积时** (第487-488行)
   ```javascript
   // 修改前
   LEFT JOIN work_records wr ON wr.t_number = m.t_number AND wr.source = 'old'
   WHERE m.source = 'old'
   
   // 修改后
   LEFT JOIN work_records wr ON wr.t_number = m.t_number AND wr.source IN ('old', 'old_api')
   WHERE m.source IN ('old', 'old_api')
   ```
