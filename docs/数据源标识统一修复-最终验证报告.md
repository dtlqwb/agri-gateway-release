## 📊 修复概览

### 问题根源
旧供应商数据存在两种 `source` 标识：
- `old`: 早期通过 Excel 导入的历史数据
- `old_api`: 后期通过 API 自动同步的新数据

但代码中多处硬编码了 `WHERE source = 'old'`，导致：
- ❌ 20台新设备未被统计到 `machines` 表
- ❌ API 同步的作业面积（约9884.61亩）未被计入
- ❌ 统计数据严重偏低

### 修复范围
共修复 **15处** SQL查询和过滤逻辑，涉及 **3个文件**：

| 文件 | 修复数量 | 说明 |

**修复后**:
