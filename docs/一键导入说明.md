## ❓ 常见问题

### Q1: 提示找不到Node.js？

**A**: 需要先安装Node.js
- 下载地址: https://nodejs.org/
- 选择LTS版本（16或18）
- 安装后重启终端

### Q2: 提示找不到.env文件？

**A**: 需要配置环境变量
```bash
# 复制示例文件
cp .env.example .env

# 编辑.env，修改数据库配置
vi .env  # 或使用其他编辑器
```

必须修改的配置：
```env
MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=你的MySQL密码
MYSQL_DATABASE=agri_gateway
```

### Q3: 导入失败，提示数据库连接错误？

**A**: 检查以下几点：
1. MySQL服务是否启动
2. .env中的数据库配置是否正确
3. 数据库 agri_gateway 是否已创建

创建数据库：
```sql
CREATE DATABASE agri_gateway CHARACTER SET utf8mb4;
```

### Q4: 可以重复运行吗？

**A**: ✅ 可以！脚本会检测重复数据，不会创建重复记录。
